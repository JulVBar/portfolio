Описание проекта

Интернет-магазин косметики на основе React. Все товары разделены на категории. Фильтраия товаров смежная - можно выбрать несколько пунктов. Фильтры по бренду и типу продукта. Так же есть фильтр по цене. В карточке товара можно выбрать цвет, добавить соответсвующий выбор в корзину, добавить в избранное. Из корзины можно удалить товары, подкорректировать количество. Продукты разных цветов добавляются в корзину по своей уникальности.
